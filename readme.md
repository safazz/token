## install & usage
# install ***Termux***
```
$ pkg install python2
$ pkg install python
$ pkg install git
$ git clone https://github.com/safazz/token
$ cd token
$ ./install
$ python2 tokenfb.py
```
# install ***Linux***
```
$ sudo apt install python
$ sudo apt install python3
$ sudo apt install git
$ git clone https://github.com/safazz/token
$ cd token
$ sudo ./install
$ sudo ./tokenfb.py
```
